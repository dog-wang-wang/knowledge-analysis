📄 code/src/algorithms.py

基础算法模块，包含查询向量构建和随机游走算法。

核心函数

query_vector(KG, query_log)

将查询日志转换为实体向量。向量的每个元素对应一个实体，值为该实体被查询的次数。

random_walk_with_restart(M, x, c, power)

带重启的随机游走算法，用于计算实体的重要性分数。公式：(I - M)^-1 = I + M + M² + M³ ...

📚 术语解释

随机游走 (Random Walk)从某节点出发，随机跳到邻居节点，重复多次。
重启概率 (Restart Probability)每次跳转时回到起点的概率。
Taylor展开用多项式近似计算矩阵逆。
大白话总结：algorithms.py提供基础图算法工具，帮助计算实体的重要性（"热度"）。
