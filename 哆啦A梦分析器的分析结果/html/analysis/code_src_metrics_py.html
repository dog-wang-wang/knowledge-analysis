📄 code/src/metrics.py

评估指标模块，计算摘要质量的F1分数、精确率和召回率。

核心函数

query_metrics(S, query)

计算单个查询在摘要上的评估指标。

total_query_log_metrics(S, query_log)

对整个查询日志计算累积的评估指标（TP/FP/FN合并计算）。

average_query_log_metrics(S, query_log)

对整个查询日志计算平均评估指标（每个查询单独计算后取平均）。

基础函数

  • precision(tp, fp, fn) - 计算精确率
  • recall(tp, fp, fn) - 计算召回率
  • f1_score(tp, fp, fn) - 计算F1分数

    • 📚 术语解释

    True Positive (TP)摘要正确返回的答案数量。
    False Positive (FP)摘要错误返回的答案数量。
    False Negative (FN)摘要遗漏的正确答案数量。
    F1分数精确率和召回率的调和平均。
    大白话总结：metrics.py是"裁判"，它评判摘要做得好不好。通过对比正确答案和摘要返回的答案，计算分数。
